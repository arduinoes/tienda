# Tailwind CSS

Tailwind Vue (opens new window)

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

Configura tailwind.config.js

module.exports = {
  content: [
    "./index.html",
    "./src/**/*.{vue,js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}

Añade el archivo src > index.css

@tailwind base;
@tailwind components;
@tailwind utilities;
